## Installation
### Windows
copy and paste the below block into a terminal
```
git clone https://github.com/jpatrick5402/HBIPS.git
cd EIPS/
.\setup.bat
```
### Linux (Debian)
copy and paste the below block into a terminal
```
git clone https://github.com/jpatrick5402/HBIPS.git
cd EIPS/
chmod +x setup.sh
./setup.sh
```
